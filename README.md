# A Desktop Pet

[中文](/README.md)|[English](/README_en.md)

一个小小的尝试用tauri,Svelte,Rive写一个可拓展的桌宠(虽然核心的插件系统还没实现

## 开始

```sh
git clone https://github.com/sevmeowple/Petti.git
cd Petti
pnpm install
pnpm tauri dev
```

构建
```sh
pnpm tauri build
```
